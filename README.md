## Car Booking Dashboard

### About Dashboard
This dashboard provides a complete analysis of car booking data, including booking trends, revenue, customer insights, vehicle performance, and trip analysis. It helps businesses monitor operations, improve customer experience, and make data-driven decisions.

### Data Source
- SQL Database
- Excel Files

### Features
- Booking and revenue analysis
- Customer and vehicle performance tracking
- Region-wise and trip-wise booking insights
- Monthly and yearly booking trends
- Interactive filters and slicers
- Dynamic KPIs and visual reports
- Multiple dashboard pages for detailed analysis
- Navigation using bookmarks and buttons
- Drill-down functionality for deeper insights
- Time intelligence calculations for trend analysis
- Responsive and user-friendly dashboard design

### Tools & Technologies
- Power BI
- SQL
- Excel
- DAX
- Power Query
- Star Schema Data Modeling
- Data Cleaning & Transformation
- Time Intelligence Functions
- Bookmarks & Navigation
- Drill Down Functionality
